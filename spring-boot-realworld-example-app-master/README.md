# Blog Backend API

This is a backend REST API project developed using Spring Boot.

## Features
- User registration and login (JWT Authentication)
- Create, update, delete blog posts
- RESTful API design
- Clean layered architecture
- MySQL database integration

## Tech Stack
- Java
- Spring Boot
- Spring Security
- MyBatis
- MySQL
- Gradle

## Project Structure
- Controller Layer
- Service Layer
- Repository Layer
- Entity & DTO

## How to Run
1. Clone the repository
2. Configure database in application.properties
3. Run the project
